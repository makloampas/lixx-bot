# lixx-bot

WhatsApp Bot powered by [Baileys](https://github.com/WhiskeySockets/Baileys)

> Generated by **Baileys Generator Bot** (Telegram)

## Fitur
- 🚫 Anti Link — Auto hapus pesan berisi link
- 🏓 Ping — Command ping/pong
- 👥 Group Mgmt — Kick, add, promote member
- ⬇️ Downloader — Download YT/IG/TikTok
- 🤖 AI Chat — Integasi AI (OpenAI/Gemini)
- 📋 Menu — List semua command
- 🖼 Sticker — Buat sticker dari gambar
- 👋 Welcome — Auto sambut member baru

## Setup

```bash
git clone https://github.com/makloampas/lixx-bot
cd lixx-bot
npm install
cp .env.example .env
# Edit .env sesuai kebutuhan
node index.js
```

## Konfigurasi
| Variable | Keterangan |
|---|---|
| PHONE_NUMBER | Nomor WA kamu (jika pakai Pairing Code) |
| OPENAI_API_KEY | API Key OpenAI |
